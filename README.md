## 🚀 Quick Start

### Prerequisites
- Node.js 18+ 
- pnpm (recommended) or npm

### Installation
```bash
# Clone the repository
git clone https://github.com/DANVERSE01/danverse-v2.git
cd danverse-v2

# Install dependencies
pnpm install

# Start development server
pnpm run dev

# Build for production
pnpm run build

# Preview production build
pnpm run preview
```

### Development
```bash
# Start dev server with host access
pnpm run dev --host

# Build and watch for changes
pnpm run build --watch
```

## 🔧 Customization

### Adding New Sections
1. Create component in `src/components/`
2. Import and add to `App.jsx`
3. Add navigation link in `Header.jsx`
4. Style with Tailwind CSS classes

### Updating Videos
1. Replace video files in `src/assets/videos/`
2. Update import paths in components
3. Ensure video format is MP4 for compatibility

### Modifying Styles
- Global styles: `src/App.css`
- Component styles: Tailwind classes in JSX
- Custom animations: CSS keyframes in `App.css`
